This is a full walkthrough for the Pokémon fangame Pokémon Reborn, using my balance patch. This walkthrough is designed to let you experience the game without hitting any major hurdles that you might during a regular playthrough, and to provide information on all of the key boss fights you will face with the rebalanced teams.

Installation

This patch needs to be installed in a specific way for it to work properly.

  1. Install Pokémon Reborn, version 19.16. (This is the version you get in the updater.)
  2. Download the All Gen Patch 6.5a and install it over the base game. Download the patch version, not the pre-patched version.
    ( Direct link to 6.5a )
  3. Download the E19 Music Pack and copy the music files into the BGM directory, as well as the single Ruby file into the Data/Mods directory. Do not copy the other remaining file.
  4. Apply any other mods you wish to use. I have a small list of recommendations below.
  5. Finally, copy my Balance Patch overwriting all files.

If you installed everything correctly, there will be a new NPC inside the starter selection room in the Grand Hall.

Routes

Reborn has a few different story routes which primarily affect events near the end of the game and in the postgame.

This walkthrough will only cover the route known as the "Bad" R-Determination path as it's the most complete. At the end of the game, you can switch to the "Good" path very easily without requiring a new save.

You can replay the game entirely to get a different route, e.g. the "Bad" Z-Regular path. That won't be covered here.

Relationship Points

This game has a relationship points system, although as far as I can tell it's pretty primitive and doesn't seem to affect that many things. I won't be covering it in this walkthrough.

Recommended Mods

Aside from the mandatory mods required for the Balance Patch, there's a few mods that I recommend for the best experience.

  • SWM Modular Modpack   - All mods - except for the starter selection mod. It doesn't seem to work though, but I don't recommend using it even if it did.
  • Extended NG+ Mod (NG++)   - essential if you're playing this game again. Less essential on the first run. Worth it purely for the Victory Road skips.
  • Pyrolusitium Z2   - overhauls the UI significantly. Make sure to install the SWM modpack compatibility file as well.

Hidden Items and Missing Events

This walkthrough will only minimally cover hidden items. I don't have the time to load up the editor and scan for the item events.

Likewise, I'll write about most event/static encounters, but I'll skip some that just really aren't worth it. You should be exploring the game as much as possible yourself, anyway!

Left vs Right

Sometimes, you might find that I tell you to go left when you actually have to go right. Sorry! I can't tell left from right. Just go the other way!

Other Useful Resources

  2. The All-Gen patch includes a spreadsheet in your game directory which contains locations for the new Gen 8 and 9 Pokémon, as well as TMs. Nearly all other information is incorrect!!!!! Prefer the information listed in this document instead.
